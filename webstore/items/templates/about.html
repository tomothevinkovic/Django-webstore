About:


This projects development has started on 20th of August 2018. Then, I had no ambition for it, neither have I thought that i will ever post it on Github and start active development. I am happy that i have changed my mind. Besides being my django programing practice project, I want this project to help everyone who is just starting out and was looking for something just like this. I am happy with the direction that the project is taking and i hope that one day, it will see the light of day. I am planning this project to release sometime in 2018.
This project is taking up most of my free time because i really like django web development. The scope of this project is huge and i really hope that one day, it will do just what it is intended for, to be the next big player on the web addvertisement and purchase!
